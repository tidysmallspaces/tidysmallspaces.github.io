# Week 01 Pin Exports

These files are upload-ready Pinterest designs at `1000 x 1500 px`.

- Upload the PNG files directly to Pinterest.
- Use the matching title, description, board, and destination from
  `content/week-01-pins.csv`.
- Keep the SVG files as editable source files. Canva Free can import them.
- Publish or schedule no more than two Pins per day.

The first batch intentionally contains original organization guidance without
Amazon product links. Add affiliate recommendations only after the Amazon
Associates account is ready.
