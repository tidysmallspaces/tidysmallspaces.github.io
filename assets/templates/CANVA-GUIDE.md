# Canva Free Design Pack

The three SVG files in this folder are sized for Pinterest at `1000 x 1500 px`.
Upload them to Canva Free as visual starting points, then create duplicates for
each Pin variation.

## Weekly workflow

1. Upload each SVG to Canva Free.
2. Duplicate the closest template for each row in `content/week-01-pins.csv`.
3. Replace the headline, list items, and website text after the final GitHub
   Pages URL is known.
4. Add an original photo, licensed Canva Free photo, or generated lifestyle
   visual when it improves the idea. Keep the text readable.
5. Export each completed Pin as PNG.
6. Schedule up to 10 Pins at a time with Pinterest's built-in scheduler.

## Style guide

- Backgrounds: `#F7F4ED`, `#FFFDF9`, and `#DCE6D8`
- Main text: `#243229`
- Accent: `#476655`
- Secondary text: `#617067`
- Use one clear promise per Pin.
- Keep layouts practical and uncluttered.
- Do not copy Amazon product images, prices, ratings, or reviews into Pins.
