# Tidy Small Spaces

Free starter site for a small-space organization content project using GitHub Pages,
Pinterest, Canva Free, and Amazon Associates.

## Publish on GitHub Pages

1. Create a public GitHub repository.
2. Upload this folder or push it with Git.
3. In the repository, open `Settings > Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the default branch and the `/ (root)` folder.
6. Wait for GitHub to show the public site URL.

## Account setup checklist

1. Confirm that the existing Pinterest account is a business account and rename
   the public username from `davidforexe` to a brand-aligned name if available.
2. Create a Canva Free account.
3. Publish the GitHub Pages site. The starter footer currently links to
   `https://www.pinterest.com/davidforexe/`; update it after the username is renamed.
4. Apply to Amazon.com Associates only after the public site has useful content.
5. Register the GitHub Pages site and Pinterest profile as promotional sites in
   Amazon Associates.
6. Replace product research placeholders only with links created inside your
   Amazon Associates account.

## Rules for content

- Stay at $0 spend until the project earns its first $30 in commissions.
- Publish two fresh organic Pins daily and batch the work weekly.
- Use original writing, original layouts, your own photos where possible, and
  licensed Canva Free assets.
- Do not copy Amazon product images, ratings, prices, or reviews into designs.
- Do not enable paid or boosted ads that send traffic to Amazon links.
- Keep the affiliate disclosure near product recommendations.

The working name `Tidy Small Spaces` is intentionally easy to replace after checking
Pinterest and GitHub availability.
