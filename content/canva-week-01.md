# Canva Week 01: First 10 Pins

Use Canva Free and create one design at `1000 x 1500 px`.

## Import the templates

1. Open Canva and select **Create a design > Custom size**.
2. Enter width `1000 px` and height `1500 px`.
3. Upload these files from `assets/templates/`:
   - `numbered-tips.svg`
   - `mistake-and-fix.svg`
   - `checklist.svg`
4. Add each uploaded SVG to a separate Canva design.
5. Duplicate the appropriate design for each Pin below.
6. Keep the website text as `tidysmallspaces.github.io`.

## Export

1. Export each completed design as PNG.
2. Name the files `PIN-001.png` through `PIN-010.png`.
3. Use the matching title, description, and destination path from
   `content/week-01-pins.csv` when publishing.
4. Schedule no more than two Pins per day.

Do not add Amazon product photos, prices, ratings, or affiliate links yet.
