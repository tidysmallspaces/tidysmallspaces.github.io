## Business-account check

On Pinterest desktop:

1. Open `Settings`.
2. Select `Account management`.
3. If you see **Convert to a business account**, the profile is still personal.
4. Convert it before publishing affiliate content so Pinterest Analytics is available.
5. Open `Settings > Edit profile`, update the name, username, bio, and website, then save.

Do not add a public contact email unless you want visitors to see it.
